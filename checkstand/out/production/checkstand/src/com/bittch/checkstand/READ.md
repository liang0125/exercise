## 项目名称
+ 超市小助手
## 项目描述
+ 基于数据库存储的电子收银台
## 项目功能
+ 基于数据库存储的电子收银台
+ 买家可以查看商品、添加订单、取消订单、修改订单
+ 卖家可以上架商品、下架商品、修改商品信息
+ 最终付款的时候可以打印订单以及总价

## 使用技术
+ Java基础知识
+ MySQL数据库
+ JDBC编程
+ 集合框架
